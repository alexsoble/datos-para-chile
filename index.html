¿Qué es Datos Para Chile?

Datos Para Chile es una comunidad que incluye desarrolladores, servidores públicos, estudiantes de informática, emprendedores y ciudadanos/ciudadanas.

Creemos que los datos abiertos pueden ser utilizados para mejorar Chile. Vamos a crear un espacio relajado, informal y estimulante para hablar, aprender/enseñar sobre los datos abiertos y conectar grupos diversos.
